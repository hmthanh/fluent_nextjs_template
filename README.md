# Fluent UI with NextJS template

## Update all package

```sh
ncu -u
```

## Installation

- Installation

```sh
npm install
```

- Run development

```sh
npm run dev
```

- Run build production

```sh
npm run build
```

- Deep check
  `npm install json`

```sh
depcheck . --json | json
```

- Sort alphabet package

```sh
npx sort-package-json
```
